## Quick Run

Clone this repo and install all dependencies:

```
pip install -r requirements.txt
```

> Sebaiknya, gunakan virtual environment.

Jalankan:

```
mkdocs serve
```

Build dokumentasi:

```
mkdocs build
```

Dokumentasi resmi:

https://www.mkdocs.org/getting-started/
